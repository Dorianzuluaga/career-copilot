# Purpose

This document defines the functional boundaries of the first version of Career Copilot.

The MVP focuses on validating the core value proposition with the minimum set of features required to solve a real user problem.

Features outside this scope are intentionally postponed to future releases.

---

# MVP Goal

Enable a user to manage an entire job application from one place.

The MVP should reduce repetitive work, improve application quality, and keep every opportunity organized.

Success is measured by the user's ability to complete the complete application workflow without relying on external tools.

---

# Target User

The MVP targets professionals actively looking for employment.

Typical users include:

- Junior Developers
- Career Changers
- Recent Graduates
- Experienced Professionals

---

# Core User Journey

The MVP should allow the following workflow:

1. User creates or uploads a Master CV.
2. User pastes a job description.
3. AI analyzes the job offer.
4. AI generates an optimized CV.
5. AI generates a personalized cover letter.
6. User saves the application.
7. User tracks the application's progress.

This is the complete value loop the MVP must validate.

---

# MVP Modules

## 1. Authentication

Users can:

- Register
- Sign in
- Sign out

---

## 2. Master CV

Users can:

- Create a Master CV
- Edit information
- Store skills
- Store work experience
- Store education

---

## 3. Job Analyzer

Users can:

- Paste a job description
- Analyze requirements
- Extract relevant skills
- Identify keywords

---

## 4. AI CV Optimizer

Users can:

- Generate an optimized CV
- Improve ATS compatibility
- Adapt content to the selected position

---

## 5. Cover Letter Generator

Users can:

- Generate personalized cover letters
- Edit generated content
- Save versions

---

## 6. Application Tracker

Users can:

- Save every application
- Track status
- Add notes
- Store company information

---

# Out of Scope

The following features are NOT part of the MVP.

- Interview Coach
- Networking
- LinkedIn Integration
- Recruiter CRM
- Salary Negotiation
- Learning Recommendations
- Career Analytics
- Browser Extension
- Mobile App
- Multi-language support
- Team Collaboration

---

# MVP Success Criteria

The MVP is successful if users can:

- Complete one application in less time than their traditional workflow.
- Generate an optimized CV.
- Generate a cover letter.
- Save and manage applications.
- Keep all information centralized.

---

# Technical Validation

The MVP should validate:

- AI integration
- Authentication flow
- Database structure
- PDF generation
- Career Relationship Management workflow

---

# Future Expansion

Future versions will progressively include:

- AI Interview Coach
- Recruiter Relationship Management
- Career Analytics
- AI Career Advisor
- Networking Management
- Learning Assistant
- Salary Insights

These features are intentionally excluded until the MVP demonstrates product-market validation.
